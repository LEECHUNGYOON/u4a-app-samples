# U4A 애플리케이션 샘플 - v1.120.21

해당 폴더는 **U4A Workspace v1.120.21** 환경에서 사용할 수 있는 U4A 애플리케이션 샘플을 제공합니다.

## 버전 정보

- U4A Workspace Version : `v1.120.21`

## 폴더 구조 예시

```text
v1.120.21/
 └─ sample/
     ├─ sample_01.mp4       # 샘플 소개 및 동작 시연 동영상
     ├─ sample_02.png       # 애플리케이션 실행 화면 이미지
     ├─ sample_03.jpg       # 애플리케이션 실행 화면 이미지
     ├─ sample.jpg          # 애플리케이션 대표 썸네일 이미지
     ├─ sample.json         # 애플리케이션 메타 정보
     └─ sample.U4A          # U4A Workspace Import용 애플리케이션 파일
```

각 샘플 폴더는 위와 같은 구조로 구성되며, 동일한 이름의 파일들이 확장자별로 제공됩니다.

## 파일 구성

| 확장자 | 설명 |
|--------|------|
| `.mp4` | 샘플 애플리케이션의 동작을 소개하는 시연 동영상 |
| `.png` / `.jpg` | 애플리케이션 실행 화면 캡처 이미지 |
| `.json` | 애플리케이션 이름, 버전 등 메타 정보 |
| `.U4A` | U4A Workspace에서 Import하여 실행 가능한 애플리케이션 파일 |

## 사용 방법

1. 원하는 샘플 폴더의 `.U4A` 파일을 다운로드합니다.
2. **U4A Workspace**를 실행합니다.
3. 상단 메뉴의 **Import** 기능을 선택합니다.
4. 다운로드한 `.U4A` 파일을 선택합니다.
5. Import 완료 후 애플리케이션을 확인합니다.

> 💡 Import 전에 동일 폴더 내 `.mp4` 및 이미지 파일로 애플리케이션 동작을 미리 확인할 수 있습니다.

## 참고 사항

- 다른 버전에서는 일부 기능 또는 화면 구성이 다르게 동작할 수 있습니다.
