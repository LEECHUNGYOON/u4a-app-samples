# U4A 애플리케이션 샘플

해당 Repository는 버전별 U4A 애플리케이션 샘플을 제공합니다.

## 폴더 구조

```text
u4a-app-samples/
 ├─ v1.0/
 │   ├─ SampleApp.U4A
 │   └─ ...
 ├─ v2.0/
 │   ├─ DemoApp.U4A
 │   └─ ...
```

각 버전 폴더 안에는 `.U4A` 확장자의 애플리케이션 파일이 포함되어 있습니다.

## 사용 방법

1. 원하는 버전 폴더로 이동합니다.
2. `[어플리케이션명].U4A` 파일을 다운로드합니다.
3. **U4A Workspace**를 실행합니다.
4. 상단 메뉴의 **Import** 기능을 선택합니다.
5. 다운로드한 `.U4A` 파일을 선택하여 Import 합니다.
